# 郑州长途客运

郑州长途客运 - 微信小程序 - 一款基于 Taro 框架开发的微信小程序

## 特性

- 支持郑州到河南各市、县之间长途汽车时刻查询
- 支持河南各地区长途汽车站地址、电话等信息查询
- 界面友好的查询服务
